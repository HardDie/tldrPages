# mpv

> Mpv player

- Change audio track
`#`

- Change subtitles
`j or J`

- Hardware acceleration
`ctrl + h`

- Use external audio
`mpv {{video.mkv}} --audio-file={{audio.ac3}}`

- Use external subtitles
`mpv {{video.mkv}} --sub-file={{sub.srt}}`

- Change speed
`[] or {}`

- Show playlist
`F8`

- Previous / Next item in playlist
`< or >`

- Volume
`9 or 0`

- Save position
`mpv --save-position-on-quit {{video}}`
