# tmux

> tmux

- Create tmux session in background
`tmux new-session -d`

- Send command in all tmux panes
`:setw synchronize-panes on`

- Rename session
`prefix + $`

- Rename window
`prefix + ,`

- Switch panes
`prefix + {`

- Switch layout
`prefix + Space`

- Switch to next / previous pane
`prefix + o`
`prefix + ;`

- Split vertical / horizontal pane
`prefix + %`
`prefix + "`
